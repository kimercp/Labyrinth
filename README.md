# Labyrinth

Labyrinth – Java Maze Game

Labyrinth is a Java-based 2D desktop game featuring procedurally generated mazes created with a randomized Depth-First Search (DFS) algorithm. The project was originally developed as an academic game development project and has been reactivated to modernize, maintain, and extend its functionality.

The game generates perfect mazes, meaning there is exactly one valid path between any two points in the labyrinth. Each level introduces a new randomly generated environment with increasing complexity, encouraging logical thinking, exploration, and problem-solving.

Features
Procedural maze generation using Randomized DFS
Perfect maze structure with a unique solution path
Scalable difficulty through progressively larger mazes
Player authentication and profile management
High-score tracking and leaderboard system
Collectible coins with score bonuses
Teleport mechanics placed in strategically selected locations
Animated game objects and sprite-based graphics
Countdown timer and level progression system
MySQL database integration for player data and scores
Custom game loop running at 60 FPS
Technical Highlights

The project demonstrates several software engineering concepts and game development techniques:

Recursive graph traversal algorithms
Procedural content generation
Object-oriented design in Java
Swing-based desktop user interface
Multithreading with timer-driven game loops
Database connectivity using JDBC
Input validation with regular expressions
Event-driven programming and game state management
Purpose

This repository serves as both a playable game and a showcase of algorithmic maze generation, Java desktop application development, and software engineering practices. The current goal of the project is to preserve, refactor, and enhance the original implementation while maintaining its educational value and retro-game charm.
